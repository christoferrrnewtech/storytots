# StoryTots — Release Notes

Google Play "What's new" notes. Limit: **500 characters per language** per
release. Paste the latest version's note into Play Console → Production (or your
test track) → Create release → Release notes.

> Reminder: each new upload must bump the version in `pubspec.yaml`
> (e.g. `version: 1.0.1+2`) — the build number (`+N`) must increase every time.
